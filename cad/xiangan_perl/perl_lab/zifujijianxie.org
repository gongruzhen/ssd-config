

|-----------+-----------------------+-------------------------------+-----------|
| shortcuts | equal character class | meaning                       | note      |
|-----------+-----------------------+-------------------------------+-----------|
| \d        | [0-9]                 | matchs for any digit          |           |
|-----------+-----------------------+-------------------------------+-----------|
| \w        | [A-Za-z0-9_]          | matchs ordinary letters,      |           |
|           |                       | digits, and underscores       |           |
|-----------+-----------------------+-------------------------------+-----------|
| \s        | [\f\t\n\r ]           | matchs whitespace             |           |
|-----------+-----------------------+-------------------------------+-----------|
| \h        | [\t ]                 | matchs horizontal whitespace  | perl 5.10 |
|-----------+-----------------------+-------------------------------+-----------|
| \v        | [\f\n\r]              | matches vertical whitespace   | perl 5.10 |
|-----------+-----------------------+-------------------------------+-----------|
| \R        |                       | matches any sort of linebreak | perl 5.10 |
|-----------+-----------------------+-------------------------------+-----------|


|-----------+--------------------+------------------------|
| shortcuts | Negating Shortcuts | equal character class  |
|           |                    | for negating shortcuts |
|-----------+--------------------+------------------------|
| \d        | \D                 | [^\d]                  |
|-----------+--------------------+------------------------|
| \w        | \W                 | [^\w]                  |
|-----------+--------------------+------------------------|
| \s        | \S                 | [^\s]                  |
|-----------+--------------------+------------------------|
